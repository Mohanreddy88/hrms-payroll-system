STEP 1: Go to Railway Dashboard

https://railway.com/dashboard

STEP 2: Create New Project
---------------------------
- Click "New Project"
- Select "Deploy from GitHub repo"
- Select: Mohanreddy88/hrms-payroll-system
- Railway will auto-build

STEP 3: Add PostgreSQL Database
--------------------------------
- In same project, click "New" → "Database" → "Add PostgreSQL"
- Railway automatically connects it (sets DATABASE_URL)

STEP 4: Set Environment Variables
----------------------------------
Click backend service → Variables tab → Add:

JWT_SECRET_KEY=YourStrongRandomSecretKey32CharsMinimum
SMTP_USER=mohan.net88@gmail.com
SMTP_PASSWORD=mlpm gqdi mvhe ufue
ASPNETCORE_ENVIRONMENT=Production

STEP 5: Run Migrations
-----------------------
- Click backend service → Shell tab
- Run:
  cd HrmsApi
  dotnet ef migrations add InitialCreate
  dotnet ef database update

STEP 6: Get Backend URL
------------------------
- Settings tab → Networking → Generate Domain
- Copy URL (e.g., https://hrms-backend.up.railway.app)
