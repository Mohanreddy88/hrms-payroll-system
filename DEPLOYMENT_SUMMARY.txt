STEP 1: PUSH TO GITHUB
----------------------
Run this command:
    cd C:\Users\HP\source\repos\walnut
    deploy-to-github.bat

OR manually:
    git init
    git add .
    git commit -m "Initial deployment"
    git remote add origin https://github.com/Mohanreddy88/hrms-payroll-system.git
    git push -u origin main

STEP 2: CREATE AZURE SQL DATABASE
----------------------------------
1. Go to: https://portal.azure.com
2. Create → Databases → SQL Database
3. Name: HrmsDb
4. Pricing: Basic (2GB) - $4.99/month
5. Copy connection string
6. Add firewall rule: 0.0.0.0 - 255.255.255.255 (allow all for testing)

STEP 3: DEPLOY BACKEND TO RAILWAY
----------------------------------
1. Go to: https://railway.com/dashboard
2. New Project → Deploy from GitHub
3. Select: Mohanreddy88/hrms-payroll-system
4. Configure:
   - Root Directory: HrmsApi
   - Start Command: dotnet run --urls=http://0.0.0.0:$PORT
5. Add Environment Variables:
   - ASPNETCORE_ENVIRONMENT = Production
   - DATABASE_URL = <Azure SQL connection string>
   - JWT_SECRET_KEY = <random 32+ character string>
   - SMTP_USER = mohan.net88@gmail.com
   - SMTP_PASSWORD = mlpm gqdi mvhe ufue
6. Run migrations:
   - Open Railway Shell
   - Run: cd HrmsApi && dotnet ef database update
7. Copy Backend URL (e.g., https://hrms-api-production.up.railway.app)

STEP 4: UPDATE FRONTEND API URL
--------------------------------
1. Open: hrms-ui/src/environments/environment.prod.ts
2. Update apiUrl to your Railway backend URL
3. Commit and push:
   git add .
   git commit -m "Update production API URL"
   git push

STEP 5: DEPLOY FRONTEND TO RAILWAY
-----------------------------------
1. Railway Dashboard → New Project
2. Deploy from GitHub → Same repository
3. Configure:
   - Root Directory: hrms-ui
   - Build Command: npm install && npm run build:prod
   - Start Command: npx serve -s dist/hrms-ui/browser -l $PORT
4. Add Environment Variables:
   - NODE_ENV = production
5. Copy Frontend URL (e.g., https://hrms-frontend-production.up.railway.app)

STEP 6: UPDATE CORS IN BACKEND
-------------------------------
1. Open: HrmsApi/Program.cs
2. Update CORS policy to include your frontend URL
3. Commit and push

STEP 7: TEST DEPLOYMENT
-----------------------
1. Open your frontend URL
2. Login with: mohanreddy77.n@gmail.com / Mohan@123
3. Test all features:
   - Dashboard loading
   - Employee management
   - Timesheet upload
   - Payroll processing
